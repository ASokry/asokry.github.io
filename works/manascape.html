Manascape

Unity 3DC#GitHub



Game Description

A wizard is trapped in a crumbling portal realm that will eventually be consumed by darkness. Help the wizard navigate through realms via portals and collect mana, while dodging enemies, to create an escape portal to a safe realm.

Controls

  • WASD/Arrow Keys to move.
  • SPACE bar to create ESCAPE portal.


Overview

Game submission for Game Maker's Toolkit Game Jame 2022. Worked with a friend for 3 days developing a Manascape(Mana + Escape). I programmed the core gameplay, scene management system, and UI in C# using Unity events, singletons, and object oriented programming.
