Game Description

A local versus, parody game based on one of Nintendo's most popular video game franchises - Super Smash Bros. Two players face off in combat on a 2D platformer. Various items and visuals reference pop cultrure icons in the Nintendo universe.

This game was developed specifically for arcade cabinets made fpr the Game Dev Club at SJSU. However, it can be played on a windows computer too. Note that you can also double jump. Hint: try falling off the stage.

Player One Controls:
  • 'AD' keys to move
  • 'W' key to jump
  • Space Bar to use action

Player Two Controls:
  • Left/Right Arrow keys to move
  • Up Arrow key to jump
  • Enter key to use action
